## 二、定价依赖全景图

```
定价决策 ──┬── B1 (Pro ¥999/月) ──→ Wave 2 定价页 + 叙事 + 对比表
           ├── B2 (Enterprise ¥298K/年咨询包) ──→ Wave 2 Enterprise 方案定义
           └── B3 (中国采购合规) ──→ Wave 2 上市路径合同/发票流程

⚠️ 如果定价高位(¥999/¥298K) vs 低位(¥299/无咨询包)，影响如下:
```

### 2.1 定价高位（¥999 Pro + ¥298K Enterprise）情景

| 影响维度 | 正面 | 负面 |
|:---------|:-----|:-----|
| **营收模型** | Pro ARPU ¥11,988/yr → 100客户 = ¥1.2M/yr | 可能流失价格敏感的小团队 |
| **品牌感知** | 符合汽车行业"专业工具"预期 | 早期获客可能慢 |
| **客户转化** | 大企业采购预算充足，Enterprise 含咨询包有竞争力 | 小团队或个人开发者转向竞品或开源版 |
| **定价页/对比表** | 三段价格梯度合理（0→¥999→¥298K） | 需准备 Enterprise 咨询包交付物 |
| **Demo/Wow Moment** | 不受影响 | 不受影响 |

### 2.2 定价低位（维持 ¥299 Pro / 无 Enterprise 咨询包）情景

| 影响维度 | 正面 | 负面 |
|:---------|:-----|:-----|
| **营收模型** | Pro ARPU ¥3,588/yr → 100客户 = ¥358K/yr（收入损失 70%） | — |
| **品牌感知** | 容易获客，低价吸引尝试 | 汽车行业可能视为"玩具工具" |
| **客户转化** | 初期快速积累用户 | 大企业 POC 无法匹配预算安全感（<¥100K/yr 太低） |
| **定价页/对比表** | 简单（免费→Pro→Enterprise 无需咨询包） | 缺乏差异化高价值方案 |
| **Demo/Wow Moment** | 不受影响 | 不受影响 |

### 2.3 不做定价决策的风险（维持现状）

| 风险 | 严重度 | 分析 |
|:-----|:------:|:-----|
| **Wave 2 无法开工** | 🔴 阻塞 | 产品叙事/A1/A2/D1/E1 全部依赖定价 |
| **客户询价无法报** | 🔴 紧急 | 任何客户接触都需要价格 |
| **创业收入模型无法验证** | 🟡 中 | 商业计划书缺少核心假设 |
| **对比表不完整** | 🟡 中 | 开源/SaaS/Enterprise 对比表缺价格列 |
| **老陈审查无法进行** | 🟡 中 | Wave 2 非正式审查没有定价页可演示 |

## 四、关键路径（Critical Path）

```
定价决策 ──→ Wave 2 (定位/对比表/上市) ──→ Wave 5 (商业推进)
    │                                                       
    └── 不阻塞 ──→ Wave 1 ──→ Wave 3 ──→ Wave 4 ──→ Wave 5 (工程侧)
                     │          │          │              
                     └── KPI启动  └── CL2冲刺  └── 竞争壁垒
```

**两条完全独立的轨道**:

| 轨道 | 依赖 | 是否受定价阻塞 |
|:-----|:-----|:--------------:|
| 🚂 工程轨道: Wave 1 → Wave 3 → Wave 4 → Wave 5 | 工程内部递进 | ❌ 不阻塞 |
| 🚂 产品轨道: 定价决策 → Wave 2 → Wave 5 | 定价先行 | ✅ 完全阻塞 |

> **策略建议**: 即使老板在 Wave 1 末仍未决策，工程轨道可独立推进至 Wave 3，不浪费一天时间。
