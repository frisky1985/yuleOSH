## 快速启动命令

```bash
# 1. 克隆 + 配置
git clone https://github.com/frisky1985/yuleOSH.git
cd yuleOSH/deploy
cp .env.production.example .env.production
# 编辑 .env.production

# 2. 启动
docker compose -f docker-compose.yml --env-file .env.production up -d

# 3. SSL
docker compose -f docker-compose.yml run --rm certbot certonly \
  --webroot -w /var/www/html \
  -d yuleosh.io --agree-tos --email admin@yuleosh.com

# 4. 验证
curl https://yuleosh.io/api/health
```
