## 2. DEF-006: MISRA Delta 模式 (L1+L2 双重)

### 验收标准
| # | 验收项 | 状态 |
|:-:|:-------|:----:|
| §14.20 | L1 仅扫描修改文件 (delta) | ✅ `mode="delta"` |
| §14.21 | L2 全量+零增量阻断 | ✅ `mode="full"` + 基线对比 |
| §14.22 | 阻断新增 Required | ✅ `_is_new_required_violation()` |

### 实现细节
- **修改**: `src/yuleosh/ci/stages.py:run_misra_check()` 新增 `mode` 参数
- **L1 delta**: `mode="delta"` → 仅扫描 `git diff HEAD~1` 修改的 .c/.cpp 文件
- **L2 full+delta**: `mode="full"` → 全量扫描 + 加载最近全量基线 + 计算新增 Required 违规
- **零增量阻断**: 新增 Required 违规优先阻断，优先级高于 fail_threshold/密度限制
- **修改**: `src/yuleosh/ci/layers.py` L1 调用传递 `mode="delta"`

### 兼容性
- 默认 `mode="auto"` 保持原有行为不变
- 全量基线从 `.yuleosh/reports/misra-trend.jsonl` 读取最近 `is_delta=False` 条目
