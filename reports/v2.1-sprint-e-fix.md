## 🟡 P1 — 缺陷逃逸率采集

### 方法
在 KPI 引擎中添加缺陷逃逸率指标：
- **定义**: 缺陷逃逸率 = 客户/测试阶段发现的缺陷 / 总缺陷数 × 100
- **采集方式**: 从 `yuleosh kpi defect-escape record` 命令记录
- **存储**: `.yuleosh/reports/defect-escape.jsonl`
- **阈值**: ≤15%

### CLI 使用
```bash
# 记录缺陷逃逸数据
yuleosh kpi defect-escape record --total=50 --escaped=3 --stage=system-test

# 查看缺陷逃逸率
yuleosh kpi defect-escape status --json

# 集成到 KPI Dashboard
yuleosh kpi status
```

### 代码改动
- `src/yuleosh/ci/kpi.py` — 新增 `record_defect_escape()`, `get_defect_escape_summary()`
- `yuleosh_cli.py` — 新增 `kpi defect-escape record|status` 子命令
- KPI status / baseline 集成
