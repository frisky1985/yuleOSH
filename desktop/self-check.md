# yuleOSH Desktop — 自测检查单

## 前置条件

- [ ] Node.js >= 18
- [ ] Python >= 3.10
- [ ] `pip install yuleosh` (或从项目 `src/` 安装)
- [ ] `frontend/out/` 已构建 (`cd frontend && npm install && npm run build`)
- [ ] `desktop/` 依赖已安装 (`cd desktop && npm install`)
