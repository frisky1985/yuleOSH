**¥0，零门槛上手。** Free 计划适合：
- 🧪 **评估 yuleOSH** — 快速体验 OpenSpec 需求管理和基础 CI 流水线
- 📂 **开源项目** — 1 个项目、每月 10 次 Pipeline 运行，满足小型开源协作需求

**限制提醒**：Free 不含 AI Code Review、无 Vector/dSPACE 适配、不支持 HIL 硬件在环。当你的项目增长到需要这些能力时，升级到 Pro 或 Enterprise。

> "先用 Free 跑通你的第一个嵌入式规范，10 次 Pipeline 足够验证概念。"

## 常见疑问

> **问：Pro 版 ¥599/月值得吗？和 Free 版差别大吗？**
> 答：Pro 版是面向嵌入式团队的完整方案，包含无限项目 + 无限 Pipeline、AI Code Review（4-Agent 并行矩阵）、SDD→DDD→TDD 全管线、一键 ASPICE 合规包、QEMU SIL 仿真、多租户隔离等。与 Free 版相比是质的飞跃——Free 适合个人评估，Pro 适合真正的团队开发。

> **问：Pro 版支持私有化部署吗？**
> 答：Pro 版支持 Docker 自托管部署（`docker-compose up`），但不包含 K8s Helm Chart 和企业级 CI/CD 集成。如需完整私有化，建议升级 Enterprise。

> **问：Enterprise 版怎么定价？有没有 ASPICE 咨询包？**
> 答：Enterprise 版 ¥98,000/年，包含私有化部署、SAML、SLA 等平台服务。目前暂不提供 ASPICE 咨询包（咨询服务如有需求可单独洽谈）。Enterprise 版支持人民币合同模板和对公转账，满足中国汽车行业采购合规流程。

> **问：Enterprise 包含几个用户？**
> 答：Enterprise 按组织计费，不限制用户数。包含最多 3 名专属工程师对接名额，额外对接名额可协商。

> **问：能先试 Pro 再决定吗？**
> 答：当然。Free 套餐已经包含基础能力。你也可以直接注册 Pro 并享受 7 天退款保证，零风险体验全部功能。
