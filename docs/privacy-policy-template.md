# 隐私政策模板（Privacy Policy）

> **用途**：yuleOSH SaaS 隐私政策，适用于 Pro / Enterprise 付费用户。
> 部署前需根据实际数据处理情况修改 `[占位符]` 内容。

---

## 1. 信息收集

### 1.1 用户主动提供的信息

- **注册信息**：邮箱地址、组织名称、用户名
- **支付信息**：通过 Stripe 处理，我们不存储信用卡信息
- **项目内容**：代码仓库、需求文档、测试用例、CI 配置

### 1.2 自动收集的信息

- **使用数据**：API 调用次数、流水线运行次数、LLM Token 用量
- **日志数据**：IP 地址、用户代理、访问时间、请求路径
- **诊断数据**：错误日志、性能指标（可选）

### 1.3 Cookie

我们使用必要的 Cookie 维护会话状态（JWT Token）。不使用第三方追踪 Cookie。

| Cookie 名称 | 用途 | 有效期 |
|:------------|:-----|:-------|
| `yuleosh_token` | JWT 认证会话 | 24 小时 |
| `yuleosh_session` | 会话标识 | 浏览器关闭 |

---

## 2. 信息用途

我们收集的信息用于：

1. **提供服务**：运行流水线、生成代码、管理项目
2. **计费**：追踪用量、处理支付
3. **安全防护**：检测异常访问、防止滥用
4. **服务改进**：分析使用模式、优化性能
5. **法律合规**：满足 ASPICE / ISO 26262 审计要求

---

## 3. 信息共享

### 3.1 第三方服务

| 服务商 | 用途 | 数据类型 |
|:-------|:-----|:---------|
| Stripe | 支付处理 | 支付信息、邮箱 |
| LLM API 提供商 | AI 辅助开发 | 需求文档、代码片段 |
| 云服务商 | 基础设施托管 | 服务器日志、存储数据 |

我们与每个第三方服务商签订数据处理协议（DPA），确保数据安全。

### 3.2 不出售用户数据

yuleOSH **不会**出售、出租或交易用户个人数据。

### 3.3 法律要求

若法律要求或善意认为必要，我们可能在以下情况下披露信息：

- 遵守法律程序
- 保护 yuleOSH 的权利或财产
- 防止欺诈或安全威胁

## 4. 数据存储与安全

### 4.1 存储位置

- **默认**：数据存储于 `[指定区域/云服务商 — TODO: 确认基础设施部署区域]`
- **企业版**：支持私有化部署，数据完全由客户控制

### 4.2 安全措施

- 所有传输数据使用 TLS 1.2+ 加密
- 数据库静态加密（AES-256）
- 访问控制：基于 JWT 的多租户隔离
- 定期安全审计和漏洞扫描
- 员工访问需经过审批和审计

### 4.3 数据保留期限

| 数据类型 | 保留期限 |
|:---------|:---------|
| 账户信息 | 账户存续期间 + 90 天 |
| 项目内容 | 项目存续期间 |
| 使用日志 | 90 天 |
| 错误诊断 | 30 天 |
| 计费记录 | 法律要求期限 |

## 5. 用户权利

根据适用的数据保护法律（包括 GDPR / 个人信息保护法），用户享有以下权利：

- **知情权**：了解我们收集和处理哪些数据
- **访问权**：请求获取您的个人数据副本
- **更正权**：要求更正不准确的个人信息
- **删除权**：在特定条件下要求删除数据
- **数据可携权**：以结构化格式获取数据
- **限制处理权**：在特定条件下限制数据处理

### 行使权利

发送邮件至 `admin@yuleosh.com`，我们将在 30 天内回复。

## 6. 数据跨境传输

- 如果用户位于中国，数据可能传输至境外服务器
- 我们采取适当保障措施（标准合同条款/安全评估）
- 企业版支持数据驻留在指定区域

## 7. 儿童隐私

yuleOSH 不面向 16 岁以下未成年人。如果我们发现无意中收集了儿童的个人信息，将立即删除。
