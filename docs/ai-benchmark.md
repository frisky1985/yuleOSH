## Benchmark 框架

Benchmark 运行器位于 `~/yuleOSH_testbench/runner.py`，支持：

```bash
# 列出所有任务
python3 runner.py --list

# 运行单个任务
python3 runner.py --task T-001

# 运行全部任务
python3 runner.py --all
```

**输出格式**: JSON + CSV，含每次运行的详细信息（耗时、代码接受率、审查通过率）。
