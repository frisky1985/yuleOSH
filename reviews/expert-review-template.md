# 专家评审模板 — Expert Review Template

> **文档**: yuleOSH Production Sprint — Phase 2 专家评审  
> **用途**: 小马 🐴 对 Phase 0/1 执行结果做正式审查 + 与老陈（行业专家）协作评审  
> **评审维度**: 代码质量 / 测试覆盖 / 部署就绪 / 架构合理 / 产品化评估  
> **通过条件**: 
> - 无 P0/P1 未关闭项
> - 综合评分 ≥80/100
> - ASPICE SWE.4/SWE.5 ≥ AL2
> - 部署验证 3 项全绿

---

## 使用说明

1. **填写评审结论**：在 Phase 0/1 执行完成后，填写各评审表头对应的"结果"列
2. **评分规则**：每维度 1-5 分（不合格=1, 需修复=2, 有条件通过=3, 通过=4, 优秀=5），权重打分
3. **证据链**：每个评分条目必须引用具体文件/行号/命令输出
4. **ASPICE 等级判定**：SWE.4（软件单元验证）和 SWE.5（软件集成与集成测试）各子项需给出 AL1~AL3 判定
